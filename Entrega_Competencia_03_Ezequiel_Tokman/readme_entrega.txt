En esta carpeta se encuentran los siguientes archivos:

- data_cleaning: script utilizado para preprocesar los datos y realizar feature engineering.
- caso1_BO: script para correr una optimización bayesiana de hiperparámetros.
- caso1_BO_excel: resultados de la optimización bayesiana.
- caso1_lightgbm: script para correr el modelo y generar los archivos de Kaggle.
- resultados_envios: planilla de Excel qeu usé para esquematizar los resultados según los distintos intenteos que realicé durante la competencia.
